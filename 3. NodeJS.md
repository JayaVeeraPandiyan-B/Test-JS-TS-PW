**Node.js** Java la Spring boot maari JavaScript ku Node

Node.js na - JavaScript ah browser ku veliya (server side) run panna use aagura runtime environment. JavaScript usually Chrome / browser la mattum run aagum Node.js vandhu adha server side kum kondu vandhadhu (Node.js is a runtime environment that allows JavaScript to run on the server side using the V8 engine)

**How it works:**
  * Chrome use pannura V8 JavaScript Engine ah use pannum
  * JS code ah machine code ah convert pannum
  * Server requests handle pannum

**What we can build using Node.js:**
  * Web servers
  * REST APIs
  * Real-time apps (chat apps)
  * Streaming apps
  * Microservices
  * Automation scripts

**Key Features:**
  * Non-blocking (Async) - Multiple requests same time handle pannum, High performance
  * Event-driven - Events base pannitu work pannum
  * Single-threaded but scalable - Thread kammi, performance adhigam
  * Fast execution - V8 engine nala

**Where mostly used?**
  * Chat apps (WhatsApp type)
  * Live tracking apps
  * Online gaming
  * Streaming platforms
  * APIs for web & mobile

**Interview short answer:** Node.js is a server-side runtime environment that executes JavaScript outside the browser, enabling developers to build scalable and high-performance backend applications.
